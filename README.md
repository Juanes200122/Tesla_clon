## 🚀 Estructura de proyecto
Dentro de su proyecto Astro, verá las siguientes carpetas y archivos:

```text
/
├── public/
│   └── CAP
│   └── CyberTruck
├── src/
│   ├── components/
│   │   └── A1_LandingModel_S.astro
│   │   └── // // //.astro
│   ├── layouts/
│   │   └── Layout.astro
│   └── pages/
│       └── index.astro
└── package-lock.json
└── package.json
└── tailwind.config.mjs
└── tsconfig.json

```
